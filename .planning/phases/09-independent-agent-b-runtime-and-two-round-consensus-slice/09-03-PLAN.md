## Phase Goal

**As a** Flow operator running `/gsd-spec-phase` and `/gsd-discuss-phase`, **I want to** have interactive questions answered by an independent Agent B running in a separate OS process with its own provider/model, **so that** delegated ask/answer is genuinely isolated, fail-closed, and evidence-backed.

<objective>
Complete the vertical slice across both workflow surfaces: run a single phase-scoped Agent B process that answers delegated questions for `/gsd-spec-phase` and then `/gsd-discuss-phase`, enforce the phase-9 scope lock (reject consensus/debate/Agora), preserve Agent A as final authority, and persist sanitized delegation evidence.

Purpose: The phase is only complete when both surfaces work end-to-end through the isolated runtime under a machine-enforced scope lock — if only one surface works, phase 9 is incomplete (SPEC R3).
Output: `agent-b-scope-lock.mjs` (deterministic scope guard), `agent-b-session.mjs` (phase-scoped start/reuse/stop), interception metadata + CLI wiring, delegation-evidence integration, and a passing both-surfaces end-to-end test.
</objective>
