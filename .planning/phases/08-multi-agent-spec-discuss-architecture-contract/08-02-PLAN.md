## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| External/provider response -> debate prototype timeout/error handling | Untrusted/late/missing answer payloads influence escalation outcomes |

| Interception/context metadata -> debate validators | Scope/version metadata may be spoofed if not checked deterministically |
