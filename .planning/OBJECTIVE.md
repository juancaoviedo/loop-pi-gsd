# OBJECTIVE

This file is the canonical extended objective artifact for Flow runs.

## Purpose

Capture the complete project description that drives bootstrap, planning, discussion, and execution.

## Allowed format variants

- OBJECTIVE.md
- OBJECTIVE.pdf
- OBJECTIVE.html

If more than one exists, the preferred precedence is:
1. OBJECTIVE.md
2. OBJECTIVE.html
3. OBJECTIVE.pdf

## Authoring guidance

- Include product intent, constraints, non-goals, success criteria, risks, and validation expectations.
- Keep a stable heading structure so traceability and context extraction are deterministic.
- Update this file when strategic project direction changes.
